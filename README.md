# Assignment Two Flutter - 2111261

This is a Flutter project created as part of the **Assignment Two** submission for student ID **2111261**. The project demonstrates a simple mobile application with a **Bottom Navigation Bar** and multiple screens such as **Home**, **Reports**, and **Profile**. It is designed to help learn Flutter UI development and navigation patterns.

---

## Features

- **Bottom Navigation Bar** with 3 tabs: Home, Reports, Profile
- **Home Screen** displaying user balance, action buttons, and recent transactions
- **Reports Screen** showing monthly spending report with a breakdown of categories
- **Profile Screen** with user information and settings menu
- Fully responsive **Flutter UI** based on Figma design
- Clean and modern design with **Material 3** styling

---

## Getting Started

These instructions will help you set up and run the project locally.

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- [Android Studio](https://developer.android.com/studio) or [VS Code](https://code.visualstudio.com/) with Flutter plugin
- A device or emulator to run the application

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/assignment_two_flutter_2111261.git

# Navigate to the project directory
cd assignment_two_flutter_2111261

# Fetch the project dependencies
flutter pub get

# Run the app
# On a connected device or emulator
flutter run

# On a specific platform (Android/iOS)
flutter run -d <device_id>
